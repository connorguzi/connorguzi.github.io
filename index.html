Projects

Laser Chaser

An autonomous laser toy for pets that uses real-time computer vision on a Raspberry Pi to detect motion and dynamically steer a servo-mounted laser to evade moving subjects. A KNN background remover is used in conjunction with a Kalman Filter to accurately detect and track animal position.

  • Computer Vision
  • Embedded Systems

Maxon EPOS ROS2 Open Source Driver

Developed an open-source ROS2 node for controlling Maxon EPOS 2/4 motors, enabling precise position and velocity control for robotics applications. Integrated both serial and CAN communication for flexible motor control.

  • Embedded Systems
  • CAN
  • ROS2 & C++

Dinger Bot

Implemented a robotic baseball hitting system using PyBullet and a policy gradient algorithm to maximize ball travel distance and accuracy. The model learned the optimal swing angle and velocity curve of the swing. When compared to a basic trajectory following algorithm the model saw around 8x improvement in ball travel distance.

  • Reinforcement Learning
  • PyBullet
  • Robotic Manipulation
